# Support for Snappy

Thank you for using Snappy.

Please use the appropriate support channel below.

---

# 🐞 Bug Reports

Found a reproducible bug?

Open a bug report on:

Issues:
../../issues

Please fill out the Bug Report template completely:

- Reproduction steps
- Expected behavior
- Environment details
- Logs/screenshots

---

# 💡 Feature Requests

Have an idea or enhancement?

Open a Feature Request:

../../issues

Use the feature request template.

---

# 💬 General Questions & Setup Help

If you're having trouble with:

- Local setup
- Redis connections
- MongoDB setup
- Docker
- Architecture questions

Please do **not** open an issue.

Check:

1. `TROUBLESHOOTING.md`

2. `DEPLOYMENT.md`

3. GitHub Discussions  
../../discussions

Community help is encouraged.

---

# 🔒 Security Vulnerabilities

Found a security issue?

Examples:

- E2E flaw
- JWT bypass
- Privilege escalation

Please follow:

`SECURITY.md`

Do NOT disclose publicly.

Email maintainers directly.
